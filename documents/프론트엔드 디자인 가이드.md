
### 계층별 책임 범위
| 레이어           | 위치               | 책임 범위                                         |
|----------------|-----------------|------------------------------------------------|
| Global (CSS)   | index.css        | 디자인 토큰(색상, 라운드, 폰트) 정의              |
| Atomic (Core)  | components/library | Radix UI + CSS 변수를 조합한 순수 UI 컴포넌트       |
| Layout (App)   | src/app          | GNB, Header, TabBar 등 전역 레이아웃 구성         |
| Business (Page)| src/pages        | 컴포넌트 배치, 데이터 페칭, gap 및 padding 결정   |

## 주요 컴포넌트

### Framework
- 디자인 스텍 : React 19, Tailwind CSS v4, Radix UI, Lucide React
- Radix UI+cn : class-variance-authority와 tailwind-merge를 사용
- Lucide React : 메뉴 아이콘은 strokeWidth={1.5}로 설정
- React Router 7 : app/router.tsx에서 정의된 경로에 따라 Layout 레이어에서 TabContainer와 Gnb가 유기적으로 작동하도록 Outlet을 적극 활용하세요.

### Design Tokens
- **Border**
  - 항상 `border-border` 사용
  - `border-black` 또는 `border-2` 사용 금지
- **Radius**
  - 컨테이너: `rounded-xl`
  - 인터랙티브 요소 (Input, Button): `rounded-md`
- **Shadow**
  - `shadow-sm`는 필요할 때만 사용

### Component Logic
- 아이콘: **lucide-react**
  - 크기: 18–20px
  - stroke-width: 2
- 클래스 병합: **tailwind-merge (cn utility)** 사용
- Radix UI primitives: 정의된 **theme tokens**로 스타일링

### Layout
- **Page spacing**: `p-6` 또는 `p-8` 사용
- **Section spacing**: `flex flex-col gap-6` 사용

## 주요 주의사항 (Avoid List)

### 배경색
- 흰색 배경(bg-white) 위에 흰색 카드(bg-white)를 올리지 마세요.
- 배경은 bg-background(slate-50), 카드는 bg-card(white)를 써서 미세한 층을 나누세요

### 아이콘
- lucide-react를 쓸 때 기본 크기가 아닌 size={18} 정도를 사용하세요

### 테이블
- table 컴포넌트에서 border-collapse를 쓰되, 행(row) 사이에는 border-b border-border만 넣으세요
- 세로줄은 가급적 뺍니다

### 포커스 링
- focus:ring-2 focus:ring-black 사용하지 말아야 합니다.
- focus-visible:ring-1 focus-visible:ring-ring 을 써서 아주 얇게 처리하세요.

### index.css 파일 수정금지
- index.css 에는 아래 내용이 작성되어있으며, 수정, 삭제, 추가를 가급적 자제 합니다
```
@import "tailwindcss";
@import url("https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@100..900&display=swap");
@theme {
    /* Typography */
    --font-sans: "Noto Sans KR", ui-sans-serif, system-ui, "Noto Color Emoji";
    /* Semantic Colors (룩앤필) */
    --color-background: #f8fafc;        /* 전체 배경: slate-50 */
    --color-card: #ffffff;
    --color-border: #e2e8f0;            /* slate-200: 절대 검정 금지 */
    --color-input: #f1f5f9;             /* slate-100 */
    --color-ring: #cbd5e1;              /* slate-300: 포커스 시 은은하게 */
    --color-primary: #0f172a;           /* 강조색: slate-900 */
    --color-muted-foreground: #64748b;  /* 보조 텍스트: slate-500 */
    /* Border Radius */
    --radius-xl: 12px;                  /* 카드용 */
    --radius-md: 6px;                   /* 버튼/인풋용 */
    /* Shadow */
    --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
}
/* Global base */
:root {
    font-family: var(--font-sans), system-ui;
    letter-spacing: -0.02em;
    font-feature-settings: "tnum";
}
```
